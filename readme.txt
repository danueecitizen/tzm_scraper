Estaba cansado de revisar la seccion de OFERTAS si ningun tipo de funcionalidad ORDENAR(SORT) de la pagina tuzonamarket.com y cree esta funcion para extraer la data de esa seccion, limpiarla  e importarla en mi propia sheet para poder ordenarla por precios DE MENOR A MAYOR!



Esta función Lambda en Python se encarga de obtener datos de una API externa, procesarlos y actualizar una hoja de cálculo de Google con los datos obtenidos. Está diseñada para ser ejecutada periódicamente o en respuesta a algún evento específico. En mi caso los datos de la API estan en formato json.

Configuración
Antes de ejecutar la función, se deben configurar ciertos parámetros, como las credenciales de Google Sheets y la URL de la API externa. Estos parámetros pueden configurarse mediante variables de entorno o directamente en el código, según sea necesario.
